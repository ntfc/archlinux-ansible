= Ansible and ArchLinux

Collection of ansible playbooks and roles used to install and maintain my personal archlinux installation.

== Requirements

Before using this you need to ensure you have already have a barebones archlinux up and running with the following:
* `python`
* `python-passlib`

Then you must ensure the following are installed if you require a specific function:
* suspend/hibernate: swap file or swap partition
*
